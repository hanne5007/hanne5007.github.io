Introduction

A morph is a simultaneous warp of the image shape and a cross-dissolve of the image colors. Image morphing was implemented to generate face morphing videos using the triangulation model and the Thin Plate Spline model.

Blending of two images was implemented in gradient domain. It is a processing technique with numerous applications such as non-photorealistic rendering, contrast enhancement, texture flattening and tone-mapping.

Please refer to my Github for more details.
